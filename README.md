### Setting up a virtual enviromnet for data analysis

1. make a virtual environment called `ml`
```
python3 -m venv ~/ml
```

2. activate the virtual environment
```
source ~/ml/bin/activate
```

3. Install required packages 
```
pip install scipy matplotlib
```
